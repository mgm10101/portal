# Claire Hand Font Setup

## Download Instructions

1. Visit https://www.dafont.com/claire-hand.font
2. Click the "Download" button
3. Extract the ZIP file
4. Copy the `.ttf` file (usually named `ClaireHand.ttf` or similar) to this folder
5. Rename it to `ClaireHand-Regular.ttf` if needed

## Font Files Needed

Place the following file in this directory:
- `ClaireHand-Regular.ttf` (or `ClaireHand.ttf`)

The CSS is already configured to use this font. Once you add the file, the font will automatically work.
